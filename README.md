# 🛡️ SOC Network Traffic Analysis Lab (TCPDump)

## 📌 Overview
This project demonstrates hands-on experience in network traffic analysis using tcpdump in a Kali Linux environment. The goal is to simulate real-world SOC analyst tasks such as packet capture, traffic filtering, and detection of suspicious activity.

---

## 🎯 Objectives
- Capture and analyze network traffic
- Identify protocols such as HTTP, DNS, and ICMP
- Detect suspicious patterns (port scanning, unusual connections)
- Document findings in a SOC-style report

---

## 🧰 Tools Used
- Kali Linux
- tcpdump
- Wireshark (optional for deeper analysis)
